Introduction
Definition
The Wayfinder application allows managers of autistic employees to create task and subtask step cards designed to assist employees progress through workplace activities and to reduce cognitive load. The application features additional pieces of helper functionality intended to address several areas of autistic difference that have the potential to cause impairment in the workplace
Research Questions
  • Can software be utilized to lessen some of the inherent neurological differences preventing autistic adults ability to function in the workplace?
  • How effective are applications that focus on e-learning, project management, and job knowledge documentation in assisting autistic users complete workplace tasks?
  • Would effective software give managers the ability to work with autistic adults, thus leading to an increase in hiring?

Anticipated Employee and Manager Goals

Employee Goals

  • Make it easier to complete work place tasks
  • Lessening of anxiety, having an easy place to review information, companion application
  • Inclusion, a tangible way to show that their team is committed to helping them succeed

Manager Goals

  • Teaching the user what to do in the workplace, giving them a knowledge base
  • Decreasing the amount of work on their end to help ASD employees succeed, path of least resistance
  • Promoting inclusion, having a tool that shows their commitment to making them feel like part of the team

User Interface Design

General UI Design Choices


  • Neutral colors
  • Simple concrete icons or images to communicate redundant information with text, and accommodate multiple ways of understanding information.
  • Make content as short as possible without sacrificing precision and specificity, to reduce cognitive burden

Navigation


  • Simple, Consistent, Prominent
  • Helper Texts to give supplementary information on each page

File System UI


  • simple, consistent
  • familiar, hopefully anyone that has used a computer will be familiar with this type of interface
